# Reduction in remaining life expectancy due to COVID-19 deaths

by [Thy Duong](https://de.linkedin.com/in/td01), Benjamin Willnecker 

supervised by [Dr. Jan Zilinsky](https://www.hfp.tum.de/digitalgovernance/team/jan-zilinsky-phd/)

This repository contains code and data from the Federal Statistical Office of Germany (Statistisches Bundesamt) for calculating the reduction in life expectency because of Covid-19 deaths per age group. It is shown, that the reduciton in remaining life expectancy from COVID-19 deaths increases with age.
